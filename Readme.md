# AirA01c

An android application to control the OLYMPUS AIR A01.

This document is written in Japanese.

-----------

AirA01C は、デジタルカメラ OLYMPUS AIR A01 にWi-Fi経由で接続し制御するアプリケーションです。

以下の機能を持っています。

- カメラのモード変更
- カメラにスマートフォンの時刻を同期
- カメラの状態取得
- カードのフォーマット
- カード内画像の全消去
- 水準器のリセット
- 水準器の0点調整
- ピクセルマッピング
- 設定の初期化
- 個別コマンドの発行
- AIR A01の操作説明

-----------
